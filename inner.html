One of my all time favorite projects was an online mobile and browser poker/bingo/pulltabs/blackjack game that I was in charge of handling everything front-end from design to coding. We choose to go the web app route so that there was only one code base for everything and built it in HTML5 canvas using the JavaScript library called CreateJS. I LOVED CreateJS because I had spent over a year of my life perfecting my ActionScript 3.0 skills just to have it killed off and CreateJS has been the closest thing I've found to the code structure and style. I am very proud of this project because we finished it in just under 4 months, given the back end only needed a few updates and the server was already finished. Well just recently I was working for a company that released a new website called NightlifeCO and we needed an app. One problem we didn't have any Objective C or Java developers on hand to do the app so I did some research and ended up choosing a neat little framework called Ionic. Ionic is built around Angular.js and has PhoneGap tied in. I studied the documentation and it took a little over a week to get a good grasp on it but lo' and behold we had an app only a couple months later.
